Multifus

Multifus est un programe pour faciliter l'utilisation des multicomptes sur le jeu Dofus.

Plus de tracas, on met le nombre de personnages par leur tour de jeu on definit les raccourcis et c'est parti!

Télécharger GitHub Donation

Sécurité

Le programme n'intéragit pas avec l'executable du jeu (Dofus.exe) de ce fait il n'est ni illégal ni banissable par quelconque méthode de sécurité car nous utilisons l'API directe de Windows.

Sans virus

Le point crucial et le plus intéressant le logiciel est distribué comme compilé sur notre site c'est à dire que vous n'avez qu'a le télécharger et l’exécuter, néanmoins le code source est disponible a quiconque souhaitant inspecter le fonctionnement du logiciel.

Stabilité

Le programme est optimisé pour utiliser le moins de ressources possible, une interface intuitive et facile d'utilisation.

Mises à jour

Le logiciel est régulièrement mis à jour non seulement pour y ajouter les fonctionalités demandés par nos utilisateurs mais égalemment pour s'assurer d'un fonctionnement 24/7.
